What is Vehicle Routing?

Vehicle Routing Problem (VRP) is a complex optimization problem where the goal is to find the most efficient routes for a fleet of vehicles to deliver goods or services to a set of customers. It helps in reducing cost, improving delivery efficiency, and minimizing travel distance. In this project, we implement a soft computing approach using Genetic Algorithms to solve the VRP. The user inputs customer locations, depot coordinates, and number of vehicles, and the app outputs optimized routes and visual representation.
